## :toolbox: Getting Started
- This Work uses npm as package manager

### :running: Run Locally

Clone the project

```bash
https://github.com/omkarsharma2821/Portfolio
```
Go to the omkar directory
```bash
cd omkar
```
Install dependencies
```bash
npm install
```
Start the server
```bash
npm start
```

### :triangular_flag_on_post: Deployment

To deploy manually 
```bash
npm run build
```
